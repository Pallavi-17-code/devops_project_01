# Orbital V4.0 - Elite CI/CD Orchestration Command Center

Welcome to the **Orbital V4.0 Command Center**. This project is a full-stack, cyber-aesthetic dashboard designed to orchestrate Continuous Integration/Continuous Deployment (CI/CD) pipelines, monitor AWS/Docker infrastructure, and act as a central hub for DevOps telemetry.

This document provides step-by-step instructions to set up and run the project on any new machine.

---

## 🛠️ Prerequisites

Before you start, ensure the following software is installed on your machine:
1. **Node.js (v18+)**: Required to run the frontend and backend. [Download Node.js](https://nodejs.org/)
2. **PostgreSQL**: Required for the backend database. [Download PostgreSQL](https://www.postgresql.org/download/)
3. **Docker Desktop**: Required to simulate infrastructure node provisioning. Ensure the Docker daemon is running in the background. [Download Docker Desktop](https://www.docker.com/products/docker-desktop/)
4. *(Optional)* **Jenkins**: If you want to test the Jenkins handshake features, Jenkins should be running locally on port `9090`.

---

## 🚀 Step 1: Database Setup

1. Open **pgAdmin** or your PostgreSQL terminal.
2. Create a new database named `cicd_dashboard`.
   ```sql
   CREATE DATABASE cicd_dashboard;
   ```
3. *(The backend will automatically create the required tables when you run the initialization script in Step 2).*

---

## ⚙️ Step 2: Backend Configuration & Startup

The backend is an Express Node.js application that handles data telemetry, Docker bridging, and Jenkins orchestration.

1. Open a terminal and navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install all required dependencies:
   ```bash
   npm install
   ```
3. Set up the Environment Variables:
   * Create a file named `.env` in the `backend` folder.
   * Add the following configuration (replace `YOUR_DB_PASSWORD` with your actual Postgres password):
     ```env
     PORT=5001
     DB_HOST=localhost
     DB_PORT=5432
     DB_USER=postgres
     DB_PASSWORD=YOUR_DB_PASSWORD
     DB_NAME=cicd_dashboard
     JWT_SECRET=super_secret_orbital_key
     JWT_EXPIRES_IN=30d
     ```
4. Run the Seed Script (This creates the database tables and the Admin account):
   ```bash
   node seed.js
   ```
5. Start the Backend Server:
   ```bash
   npm run dev
   ```
   *You should see the "Orbital V4.0 Command Core Online" banner in your terminal if successful.*

---

## 🖥️ Step 3: Frontend Configuration & Startup

The frontend is a high-performance React application utilizing Vite and Tailwind CSS.

1. Open a **new, separate terminal** and navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install all required dependencies:
   ```bash
   npm install
   ```
3. Start the Frontend Development Server:
   ```bash
   npm run dev
   ```
4. The terminal will provide a local link (usually `http://localhost:5173`). `CTRL + Click` the link to open the dashboard in your browser.
